# Skill Manager v1.0.0

## 🎉 首个正式版本

这是 Skill Manager 的首个正式版本，一个用于管理 Claude Code 技能的桌面应用程序。

## ✨ 主要特性

### 🚀 纯 Rust 架构
- ✅ 完全独立的可执行文件
- ✅ 无需 Node.js 或其他运行时
- ✅ 零依赖，开箱即用

### 🔒 安全扫描
- ✅ 内置安全规则引擎
- ✅ 自动检测危险代码模式
- ✅ 9 大类核心安全规则
- ✅ 风险评分和阻断机制

### 📦 技能管理
- ✅ GitHub 一键导入
- ✅ 本地技能安装
- ✅ 系统级和项目级技能支持
- ✅ 技能扫描和查看

### 🌐 远程初始化
- ✅ 首次启动自动下载配置
- ✅ 可预装常用技能
- ✅ 配置在线更新

## 📦 下载

### Windows (x64)
- **MSI 安装包**: `Skill Manager_1.0.0_x64_en-US.msi` (推荐)
- **绿色版**: `skill-manager.exe` (单文件)
- **初始化数据**: `init-data.zip` (自动下载)

### 文件说明

| 文件 | 大小 | 说明 |
|------|------|------|
| skill-manager.exe | ~10MB | 单文件可执行程序，无需安装 |
| Skill Manager_1.0.0_x64_en-US.msi | ~3.5MB | Windows 安装包 |
| init-data.zip | ~750B | 初始化配置和数据 |

## 🚀 快速开始

### 方式 1: 使用 MSI 安装包（推荐）
1. 下载 `Skill Manager_1.0.0_x64_en-US.msi`
2. 双击安装
3. 从开始菜单启动

### 方式 2: 使用绿色版
1. 下载 `skill-manager.exe`
2. 放到任意目录
3. 双击运行

**注意**: 首次启动需要网络连接，会自动从 GitHub 下载初始化数据到 `C:\Users\你的用户名\.skills-manager\`

## 📂 目录结构

应用程序会在用户目录创建以下结构：

```
C:\Users\你的用户名\.skills-manager\
├── config.json              # 应用配置
├── marketplace.json         # 市场数据
└── skills\                  # 已安装的技能
    └── example-skill\       # 示例技能
        └── SKILL.md
```

## 🔧 系统要求

- **操作系统**: Windows 10/11 (x64)
- **网络**: 首次启动需要网络连接
- **权限**: 需要用户目录写权限

## 📝 使用说明

### 导入技能

**从 GitHub 导入**:
1. 点击"导入技能"
2. 选择"GitHub"
3. 输入仓库地址，如 `username/repo`
4. 点击导入

**从本地导入**:
1. 点击"导入技能"
2. 选择"本地"
3. 选择技能目录
4. 点击导入

### 安全扫描

应用会自动对导入的技能进行安全扫描：
- 🟢 安全评分 75+ : 低风险
- 🟡 安全评分 50-74: 中等风险
- 🟠 安全评分 20-49: 高风险
- 🔴 安全评分 <20 或触发严重规则: 阻止安装

## 🛠️ 技术栈

- **后端**: Rust + Tauri 2
- **前端**: React 19 + TypeScript
- **UI**: DaisyUI + TailwindCSS
- **状态管理**: Zustand
